# Hangman Game 🎯
A Python implementation of the classic Hangman word-guessing game.  

---

## 📖 Description
This project demonstrates:
- Loops and conditional logic
- Lists and string manipulation
- Game state tracking (lives, guessed letters)
- ASCII art for visual feedback

---

## 🎮 Features
- Random word selection
- Lives system (lose a life for each wrong guess)
- ASCII art hangman stages
- Interactive console gameplay
